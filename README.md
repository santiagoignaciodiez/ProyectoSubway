# ProyectoSubway
Gestión de asistencia empleados de subway


Intalar:
pip install django
pip install django-cors-headers
pip install djangorestframework      # Si usás APIs REST
pip install pillow                   # Si manejás imágenes (por ejemplo, logos)
pip install python-decouple           # Para variables de entorno (configuración segura)
pip install psycopg2-binary           # Si usás PostgreSQL
pip install mysqlclient               # Si usás MySQL
python -m pip install Pillow
